Spatial Decision Support System for Tsomgo Lake Monitoring

WebGIS-based Spatial Decision Support System for lake area dynamics, water temperature and water quality monitoring of Tsomgo Lake, Sikkim.

Study Area

Tsomgo Lake is a high-altitude glacial lake located in East Sikkim, India, at an elevation of approximately 3,753 metres.

This DSS integrates waterbody extraction, temporal lake area monitoring, and spatial visualization of water quality indicators such as surface water temperature, pH, Total Dissolved Solids (TDS), and Electrical Conductivity (EC) for Tsomgo Lake between 2017 and 2025.

Machine Learning Model Comparison

Model comparison was performed using cubic spline interpolated pseudo-data.

Model R² RMSE MAE
Linear Regression 0.0798 0.003125 0.002543
Polynomial Regression 0.1072 0.003078 0.002480
Random Forest 0.7631 0.001585 0.001134

Random Forest achieved the highest R² and lowest error values, indicating superior performance for modelling the Lake Area–LST relationship using interpolated pseudo-data.

Prediction Note

These projections are based on linear trend estimation using observed data from 2017–2025. They indicate possible trend direction and should not be interpreted as precise forecasts.
